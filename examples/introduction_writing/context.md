# Parent Context Definitions

## Python
Python is a high-level, general-purpose programming language known for its readability and wide adoption in software development, data science, and AI/ML applications. It supports multiple programming paradigms including object-oriented, functional, and procedural programming.

## Agent
An agent is an autonomous software entity that can perform tasks, make decisions, and interact with its environment or other systems to achieve specific goals. In the context of AI and Claude Code, agents are specialized subprocesses that handle complex multi-step tasks autonomously with dedicated tools and capabilities.

## Skill
A skill is a packaged, reusable capability or workflow that extends the functionality of an AI assistant or agent system. Skills define structured procedures for handling specific types of tasks, such as code review, documentation generation, or project management, and can be invoked on demand to perform specialized work.

## Harness
A harness is a framework or set of tools that provides structure, standardization, and automation for executing and validating tasks. Harnesses abstract away repetitive boilerplate code, enforce consistent patterns, and provide built-in utilities for common operations like verification, error handling, and dependency management.
